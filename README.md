# Sistema de Reportes

Este proyecto genera reportes HTML dinámicos a partir de datos provenientes de un archivo Excel o un JSON equivalente. El objetivo es ofrecer un panel moderno para marketing digital.

## Requisitos
- Python 3.11+

## Instalación
```bash
pip install -r requirements.txt
```

## Uso
1. Coloca tu archivo Excel en la raíz o proporciona un JSON con la estructura esperada.
2. Ejecuta:
```bash
python -m src.app tu_archivo.xlsx
```
El comando anterior generará un archivo `reporte.html` con el panel.
